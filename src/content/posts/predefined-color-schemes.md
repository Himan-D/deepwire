Lipi includes multiple predefined color schemes.

Each theme focuses on readability and restrained contrast.

## Paper

A warm editorial theme inspired by printed books and journals.

```css
--background: #F5F4ED;
--foreground: #22201C;
--accent: #7B5C42;
```

## Forest

A muted green theme focused on long reading sessions.

```css
--background: #EFF2EB;
--foreground: #1E241D;
--accent: #5D7555;
```

## Midnight

A darker cinematic theme for low-light environments.

```css
--background: #141413;
--foreground: #ECE6DA;
--accent: #C8A07A;
```

## Switching Themes

Themes are controlled using the `data-theme` attribute.

```html
<html data-theme="midnight">
```

## Philosophy

Lipi avoids highly saturated palettes.

The goal is to create:

- calm reading environments
- strong typography contrast
- softer visual hierarchy
